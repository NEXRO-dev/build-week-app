# Echly Load Signal v1

## 位置づけ

Echly Load Signal v1は、夜のチェックイン時点における「その日に経験した主観的ワークロード」と「現在の眠気」を0〜100で要約し、翌日の予定調整を支援する非診断指標である。

ストレス、うつ、バーンアウト、疾患を判定するものではない。音声だけから絶対的な心理負荷を推定する手法には、実環境で一貫した精度を示す十分な根拠がないため、自己評価を主情報、音声を個人内変化の補助情報として扱う。

## 先行研究

### 主観的ワークロード

NASA-TLXは、精神的要求、身体的要求、時間的要求、パフォーマンス、努力、フラストレーションの6次元で課題後の主観的ワークロードを測る。20年間で広範な領域に利用され、実験上重要な操作に対する感度が報告されている。

- [Hart, 2006, NASA-TLX; 20 Years Later](https://doi.org/10.1177/154193120605000909)
- [NASA Task Load Index v1.0 manual](https://ntrs.nasa.gov/citations/20000021487)
- [NASA報告: Raw TLXと重み付き得点の相関0.96〜0.98](https://ntrs.nasa.gov/api/citations/20020021642/downloads/20020021642.pdf)

本実装は6項目の単純平均であるRaw TLXを使用する。ただし、NASA-TLXは本来、特定課題の実行直後に用いる尺度であり、「一日全体」への適用はEchly独自の拡張である。

### 眠気

Karolinska Sleepiness Scale (KSS)は現在の眠気を1〜9で自己評価する。KSSの上昇は反応時間、注意の脱落、EEGのα・θ活動と有意に関連し、眠気指標としての妥当性が示されている。

- [Kaida et al., 2006, Validation of the KSS against performance and EEG variables](https://pubmed.ncbi.nlm.nih.gov/16679057/)

### 音声特徴

発話速度、休止率、休止頻度は認知負荷分類に利用できる可能性が報告されている。一方、ストレスに伴う話速や音声品質の変化方向は研究間で一貫せず、マイク、言語、性別、発話課題、個人差の影響を受ける。

- [Gorovoy et al., 2010, speech rate / pause rate / pause percentageによる認知負荷分類](https://proceedings.cmbes.ca/index.php/proceedings/article/view/538)
- [Vocal-Stress Diary, 日常の仕事ストレッサーと音声の縦断研究](https://doi.org/10.1177/09567976211068110)
- [2025 systematic review: 実環境での音声指標には不一致と変動がある](https://pubmed.ncbi.nlm.nih.gov/40705747/)

このため、音量の絶対値は得点に使用しない。発話速度と内部休止率も集団平均との差ではなく、同じ利用者の過去記録との差だけを使う。

## 算出方法

### 1. Raw TLX

各項目を0〜100、5点刻みで回答する。

- 精神的な要求
- 身体的な要求
- 時間的な切迫
- 達成度への不満
- 必要だった努力
- 不安・いらだち

```
Raw TLX = 6項目の単純平均
```

パフォーマンス項目は、画面上では「達成度への不満」とし、高いほど負荷方向になるよう統一する。

### 2. KSS

```
KSS正規化値 = (KSS - 1) / 8 * 100
```

### 3. 音声の個人内変化

20秒以上で発話速度と内部休止率が取得できた過去記録を有効記録とする。5件未満では音声得点を使用しない。

現在値と過去中央値の差を、Median Absolute Deviation (MAD)で標準化する。MADがほぼ0の場合は中央値の15%を代替スケールとする。通常変動として1スケール分を無得点にし、それを超える変化量を0〜100へ制限する。

開始・停止操作による無音は休止率から除外する。無音閾値は各録音のRMS分布から適応的に決める。

### 4. 合成

音声ベースラインなし:

```
Score = Raw TLX * 0.85 + KSS * 0.15
```

音声ベースラインあり:

```
Score = Raw TLX * 0.75 + KSS * 0.15 + 音声変化 * 0.10
```

重みは先行研究から直接導出された係数ではない。自己評価を主、眠気を安全側の補助、音声を小さな個人内補助とする設計仮説である。

## 表示区分

- 0〜39: 低〜通常
- 40〜59: やや高い
- 60〜100: 高い

これらは臨床カットオフでもNASA-TLX公式区分でもない。翌日の予定削減や休息提案を開始するための製品上の境界である。

## 検証研究案

Echly独自部分を検証するには、次の前向き縦断研究を想定する。

- 対象: 就業成人100人以上
- 期間: 8週間以上、1人あたり20記録以上
- 入力: Echlyスコア、Raw TLX、KSS、音声特徴
- 外部基準: 翌日のタスク未完了率、予定変更数、翌朝の回復感、可能ならPVT
- 解析: 個人固定効果を含む混合効果モデル、時系列を分離した交差検証
- 比較: Raw TLX単独、Raw TLX+KSS、Raw TLX+KSS+音声
- 事前登録: 重み、除外基準、主要評価項目、40/60境界
- 成功条件: 音声追加モデルが個人外ではなく個人内予測で再現性のある増分妥当性を示すこと

音声追加で改善しなければ、音声10%は廃止しRaw TLX+KSSへ戻す。利用者ごとの音声・自己評価データは健康関連情報に準じて扱い、保存期間、削除、同意を明示する必要がある。
